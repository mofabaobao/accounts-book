# 记账本 (Accounts Book)

一个简单的本地记账应用，支持网页端查看和管理消费记录。

## 功能特点

- ✅ 自动识别记账消息
- ✅ 支持支出/收入分类
- ✅ 日期指定
- ✅ 网页报表展示
- ✅ 分类筛选
- ✅ 时间筛选（今天/本周/本月/本年）
- ✅ 图表统计（饼图 + 折线图）
- ✅ 页面直接删除记录

## 分类预设

- 餐饮
- 交通
- 购物
- 工资
- 退款
- 房租
- 通讯
- 医疗
- 其他

## 使用方法

### 1. 记账

在飞书/微信对话中告诉我：
```
记账 餐饮 30元
记账 工资 10000元
记账 2026-03-01 交通 20元
```

### 2. 查看报表

在浏览器中打开 `accounts.html` 文件：
```
data/accounts.html
```

### 3. 删除记录

在网页中点击对应记录的「删除」按钮即可。

## 文件结构

```
data/
├── accounts.json   # 数据存储（JSON格式）
└── accounts.html   # 报表展示页面
```

## 数据说明

- 数据存储在浏览器 localStorage
- 同一浏览器内数据持久化
- 更换浏览器需重新记录

## 技术栈

- 纯 HTML + CSS + JavaScript
- Chart.js 图表库
